# NRKSubParser

NRKSubParser is a Python library/tool for dealing with malformed TTML subtitles from [NRK](https://tv.nrk.no)

## Installation

Copy the NRKSubParser to your $PATH location of choice


## Usage

```bash
NRKSubParser broken.subtitles.s01e01.ttml
```
